Description

This was a solo coding project that I completed for CS 128 - Intro to Computer Science II. We were tasked with implementing multiple C++ classes that generated a grayscale topography map from a dataset of elevation values, stored the values in arrays, ran a pathfinding algorithm over the data to find paths of least elevation change from every possible starting point on one edge, and output a visualization of those paths on the topography map (colored red for each possible starting point, green for the overall path of least change).

Users could input their own .dat file (which could be sourced from NASA or other organizations online), and specify an output file to view the visualized results.
